# CryoLock 🔐

Lock crypto. Stream unlocks in real-time.

CryoLock is a full-stack Web3-inspired dApp that simulates token vesting and real-time streaming unlocks.

## 🚀 Features
- Token locking with custom duration
- Real-time streaming unlock logic
- Live dashboard with progress tracking
- Withdraw unlocked tokens anytime
- Clean modern UI (Web3 inspired)
